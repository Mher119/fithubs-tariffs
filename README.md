# Advort - Landing Page Layout

Тестовое задание на позицию HTML Developer. Верстка адаптивного лендинга с использованием современных технологий.

## Технологии

* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Icons:** Custom SVG (Figma optimized)
* **Deployment:** [Vercel](https://vercel.com/)

## Особенности реализации

* **Адаптивность:** Полная поддержка мобильных (Mobile-first) и десктопных версий.
* **Pixel Perfect:** Максимально точное соответствие макету Figma.
* **Интерактивность:** Реализована логика выбора тарифных планов и кастомные стилизованные чекбоксы.
* **Оптимизация:** Использование Next.js Image и SVG для обеспечения высокой производительности.

## Установка и запуск

1.  Клонируйте репозиторий:
    ```bash
    git clone [https://github.com/Mher119/html_testing.git](https://github.com/Mher119/html_testing.git)
    ```
2.  Установите зависимости:
    ```bash
    npm install
    ```
3.  Запустите сервер разработки:
    ```bash
    npm run dev
    ```

## Структура проекта

* `/app` — основные страницы и серверные компоненты.
* `/public` — статические ресурсы (изображения, иконки).
* `/components` — UI-компоненты (Cards, Checkbox, Layout blocks).
